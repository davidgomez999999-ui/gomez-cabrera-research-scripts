# gomez-cabrera-research-scripts

Research code repository by **David Gómez Cabrera**.

I am a social researcher with a background in Social and Cultural Anthropology, specialising in Data Science applied to Social Sciences. My work focuses on the analysis of human behaviour on digital platforms, with particular interest in the relationship between algorithmic architecture and the production of cultural meaning. I operate at the intersection of computational methods and social theory.

This repository contains the scripts, pipelines, and analytical tools developed across my research projects. Each project is self-contained in its own directory, with its own README detailing data sources, pipeline structure, and citation instructions.
